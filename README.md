# rtsp_stream_to_pyaudio
the purpose of this script is to monitor a rtsp stream for volume spikes and then start recording the audio with pyaudio.
usecase might be a babyphone or other sound monitoring.

tested with the rtsp stream of a tp-link tapo C110

this script will be a central part of a babyphone application which sends automated messages via signal-cli with the recorded sound files as an attachment.
